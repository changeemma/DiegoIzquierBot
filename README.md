# diego-izquierbot

## Requerimentos
- Instalar claves SSH con el servidor pfSense.
- Python 3.6 o mayor
- pip3
- Telegram API Token

## Instalación de librerías
```bash
pip3 install -r requirements.txt -e .
```
### Notas adicionales

Se deben generar los archivos `*_cfg.py` en base a los archivos `*_cfg.py.example`
